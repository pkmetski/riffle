# Figure annotations â€” implementation plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Annotate figures (raster images, inline SVG, `<figure>`, `<picture>`) in EPUBs â€” captured by long-press or absorbed into text-selection ranges â€” and render them full-size with captions in the Highlights-mode elided reader.

**Architecture:** Add a new `TYPE_IMAGE` annotation and one new nullable column (`embeddedFigures`, JSON) to the existing `AnnotationEntity` table via a Room 43â†’44 migration. Extend the existing `FigureTapScript` (which already detects the "zoomable" element set) with a long-press handler and a range-scan helper, so the annotatable set exactly equals the zoomable set. Extend `HighlightsPublicationFactory` to emit `<figure>` blocks for image annotations and to interleave figures inside text-highlight ranges. Draw a coloured border decoration on annotated figures in the normal reader. Extend the Web Annotation sync body with a `riffle:image` type.

**Tech Stack:** Kotlin, Room, Readium 3.3.0 (`EpubNavigatorFragment` + `ContinuousReaderView`), Coil (used elsewhere; here only the SVG source is stored, no Coil SVG decoder needed for v1), Hilt, Compose.

## Global Constraints

- **Reader modes.** Every capture-side and visual-side change must work in all three reader modes: paginated, vertical, continuous (CLAUDE.md).
- **Migration rules.** Every new Room migration bumps the `@Database` version, adds an exported schema JSON at `core/database/schemas/com.riffle.core.database.RiffleDatabase/44.json`, is registered in `DataModule.kt`'s `addMigrations(...)`, and has a step test *plus* an entry in `migrateFullChain` in `core/database/src/androidTest/kotlin/com/riffle/core/database/MigrationTest.kt` (CLAUDE.md).
- **Constants, not literals.** All annotation-type checks reference `AnnotationEntity.TYPE_IMAGE`, never the string `"IMAGE"` (CLAUDE.md).
- **Regression tests are mandatory.** Every task's final step names the assertion that would flip red if the fix were reverted (CLAUDE.md).
- **Logging.** Any new production log lines use `logger.d(LogChannel.X) { â€¦ }` â€” never `Log.d("RIFFLE_*", â€¦)` (CLAUDE.md).
- **Do not build/install APKs unless the user explicitly asks.** JVM tests + `make harness-test` are the validation gates (CLAUDE.md).

```kotlin
object FigureCaptionWalker {
    val CAPTION_RESOLVER_JS = """
        function resolveCaption(el) {
            if (!el) return "";
            var fig = el.closest ? el.closest('figure') : null;
            if (fig) {
                var cap = fig.querySelector('figcaption');
                if (cap && cap.textContent) return cap.textContent.trim();
            }
            var alt = el.getAttribute && el.getAttribute('alt');
            if (alt) return alt;
            var aria = el.getAttribute && el.getAttribute('aria-label');
            if (aria) return aria;
            return "";
        }
    """.trimIndent()

    val SVG_SERIALIZER_JS = """
        function serializeSvg(svg) {
            if (!svg) return null;
            try { return new XMLSerializer().serializeToString(svg); } catch (e) { return null; }
        }
    """.trimIndent()

    val FIGURES_IN_RANGE_JS = """
        $CAPTION_RESOLVER_JS
        $SVG_SERIALIZER_JS
        function figuresInRange(startNode, endNode) {
            var range = document.createRange();
            range.setStartBefore(startNode);
            range.setEndAfter(endNode);
            var walker = document.createTreeWalker(range.commonAncestorContainer, NodeFilter.SHOW_ELEMENT, {
                acceptNode: function(n) {
                    var tag = (n.tagName || "").toLowerCase();
                    if (tag !== 'img' && tag !== 'svg' && tag !== 'picture' && tag !== 'figure') return NodeFilter.FILTER_SKIP;
                    if (!range.intersectsNode(n)) return NodeFilter.FILTER_REJECT;
                    return NodeFilter.FILTER_ACCEPT;
                }
            });
            var out = []; var seen = new Set(); var order = 0; var node;
            while ((node = walker.nextNode())) {
                var target = node.tagName.toLowerCase() === 'figure'
                    ? (node.querySelector('img') || node.querySelector('svg') || node.querySelector('picture'))
                    : node;
                if (!target || seen.has(target)) continue;
                seen.add(target);
                var tag = target.tagName.toLowerCase();
                var entry = { caption: resolveCaption(target), order: order++ };
                if (tag === 'svg') { entry.svg = serializeSvg(target); entry.href = null; }
                else if (tag === 'picture') {
                    var img = target.querySelector('img');
                    entry.href = img ? img.getAttribute('src') : null; entry.svg = null;
                } else { entry.href = target.getAttribute('src'); entry.svg = null; }
                out.push(entry);
            }
            return out;
        }
    """.trimIndent()
}
```

```kotlin
data class FigureLongPressPayload(
    val kind: String,

class FigureTapBridge(
    private val onZoom: (FigureZoomPayload) -> Unit,
    private val onLongPress: (FigureLongPressPayload) -> Unit,

    @JavascriptInterface
    fun onFigureLongPress(json: String) {
        val obj = JSONObject(json)
        onLongPress(FigureLongPressPayload(
```

```kotlin
object FigureBorderDecoration {
    fun buildCssRules(annotations: List<Annotation>): List<String> {
        data class FigureRef(val selectorKey: String, val color: String, val updatedAt: Long)
        val refs = mutableListOf<FigureRef>()
        for (a in annotations) {
            val color = a.color
            when (a.type) {
                AnnotationEntity.TYPE_IMAGE -> a.imageHref?.let { refs += FigureRef(it, color, a.updatedAt) }
                AnnotationEntity.TYPE_HIGHLIGHT -> a.embeddedFigures?.forEach { f ->
                    f.href?.let { refs += FigureRef(it, color, a.updatedAt) }
                }
            }
        }
        return refs.groupBy { it.selectorKey }
            .mapValues { (_, group) -> group.maxByOrNull { it.updatedAt }!! }
            .values
            .map { r -> "img[src\$=\"${r.selectorKey}\"], picture:has(img[src\$=\"${r.selectorKey}\"]), figure:has(img[src\$=\"${r.selectorKey}\"]) { outline: 2px solid ${r.color.toCssRgba()}; outline-offset: 2px; }" }
    }
}
```
